# Калькулятор расходов (семейный бюджет)

Веб-приложение для учёта доходов и расходов семьи: фильтры, графики, лимиты по категориям на текущий месяц, экспорт CSV/JSON и импорт резервной копии. Данные хранятся в **localStorage** браузера.

### Без установки программ (рабочая версия)

Основной файл приложения без сборки — **`family-budget-fixed-with-photos.html`**. Откройте его двойным щелчком в браузере (Chrome, Edge, Firefox и т.д.); Node.js не нужен.

Фото участников лежат **в той же папке**, что и HTML: `Вика.png`, `ГришА.png`, `Алиса1.JPG`, `Алина.png`. При переносе на другой компьютер копируйте эти файлы вместе со страницей.

Данные в браузере — ключ **`family-budget-app-v4`** (как у версии на React). Запасной вариант без привязки к файлам картинок: `budget-offline.html`.

## Возможности

- Учёт доходов и расходов по участникам и категориям (включая свою статью).
- Дашборд: круговая диаграмма расходов, столбцы по участникам, динамика по месяцам.
- Лимиты по статьям расходов с полосой прогресса и процентом использования.
- История операций с поиском и фильтрами.
- Подсказки по бюджету на основе баланса и лимитов.
- Экспорт в CSV и JSON, импорт JSON из этого же приложения.

## Стек

- React
- [Recharts](https://recharts.org/) — графики
- [Lucide React](https://lucide.dev/) — иконки
- Стилизация через классы **Tailwind CSS** (`className`, утилиты вроде `rounded-3xl`, `bg-slate-950`).

## Структура проекта

| Файл / папка | Назначение |
|--------------|------------|
| **`family-budget-fixed-with-photos.html`** | **Рабочая офлайн-страница** (всё в одном файле + фото рядом) |
| `budget-offline.html` | То же приложение, без файлов фото (инициалы вместо снимков) |
| `index.html` | Точка входа HTML, контейнер `#root`, подключение `main.jsx` |
| `main.jsx` | Рендер React и импорт стилей |
| `App.jsx` | Основное приложение |
| `index.css` | Директивы Tailwind |
| `public/favicon.svg` | Иконка вкладки |

Офлайн-версия с фото подгружает снимки из текущей папки; упрощённая **`budget-offline.html`** рисует только инициалы.

Сборка React: [Vite](https://vitejs.dev/) + [@vitejs/plugin-react](https://github.com/vitejs/vite-plugin-react).

## Запуск

```bash
npm install
npm run dev
```

Откройте в браузере адрес из терминала (обычно `http://localhost:5173`). Сборка для продакшена: `npm run build`, предпросмотр: `npm run preview`.

## Данные и ключ хранилища

Состояние сохраняется под ключом `family-budget-app-v4` в формате `{ transactions, limits }`. Очистка данных сайта в браузере удалит и бюджет.
